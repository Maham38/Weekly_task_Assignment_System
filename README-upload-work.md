# Upload Work Module — SafeX Weekly Task Assignment System

Individual Week 2 contribution by Ayesha Kamran (Group 50). Lets interns submit weekly deliverables (a work link + optional notes) against an assigned task, and view their own submission status (Pending / Submitted / Late) per task.

## Tech Stack
- Frontend: React (Vite)
- Backend: FastAPI
- Database: Neon (PostgreSQL) — shared across all Group 50 modules
- API: REST
